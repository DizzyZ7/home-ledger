# Screenshot assets

Add redacted product screenshots here before publishing a release or portfolio post.

Recommended files:

- `dashboard.png` — inventory dashboard with attention indicators.
- `item-detail.png` — an item card with warranty and maintenance context.
- `item-receipts.png` — the receipt attachment section with synthetic filenames only.
- `maintenance.png` — the maintenance workflow or history screen.

Use mock mode or synthetic local demo data. Do not include real addresses, serial numbers, receipt numbers, purchase prices, account details, tokens, or personal photos.
